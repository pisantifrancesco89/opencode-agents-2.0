# Quick Start - 3 Steps

## Step 1: Clone

```bash
git clone https://github.com/pisantifrancesco89/opencode-agents-2.0.git
```

## Step 2: Install (One Time)

```bash
./opencode-agents-2.0/setup.sh
```

This installs the skill in OpenCode.

## Step 3: Setup Your Project

```bash
cd ~/my-project
/opath/to/opencode-agents-2.0/setup.sh ~/my-project
```

Then talk to the AI:

```
I want to create a SaaS for X
```

---

## That's It!

The AI will:
1. Ask you questions
2. Create a plan
3. Build a team of agents
4. Execute tasks
5. Verify quality
6. Update memory
7. Deliver results
